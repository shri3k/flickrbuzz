# FlickrBuzz Server

Uses [fastify](https://fastify.dev/) as http framework with [typebox](https://github.com/sinclairzx81/typebox) as type provider

## Start application

- [Install pnpm](https://pnpm.io/installation)

**Install packages**
```bash
pnpm install
```
**Start server**
```
pnpm dev
```

## Unit tests

```bash
pnpm test
```
